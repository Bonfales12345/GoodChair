# GoodChair

A basic launcher for Android.

## Description
GoodChair is a minimalist launcher designed for Android devices, focusing on simplicity and ease of use.

## Features
- Lightweight and fast.
- Clean, clutter-free interface.
- Basic app management.

## Usage of AI in this project

During development, Gemini in Android Studio was utilized as an AI coding assistant. It helped streamline the workflow by accelerating bug fixes, optimizing code structure, and cleaning up messy code.
